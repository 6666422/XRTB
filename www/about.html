Open Source

This is an implementation of the openRTB as an open source project. It includes an RTB engine written in JAVA 1.8 and an administration system written in Ruby On Rails.

RTB Bidding Engine

This project contains 3 major components 1) A Real Time Bidding engine; 2) A Simulator for sending test bids to the bidder; 3) Administration system. The RTB comes out of the box ready to run in a multi-server bidding farm, you just need to lash it up and administer it.

Exchange Simulator

A simple exchange is provided so that you can develop and test your own ads. A sample JavaScript snippet is included in your web page that contacts the exchange, which then sends a bid to the RTB4FREE bidding engine. Based on the campaigns loaded, the bidder makes an appropriate bid response. The exchange sends the win notice, and the ad is returned all the way through the system.

When you load your test page, the RTB magic happens and the ad is returned on the page. Clicking the ad causes the click through processing to be initiated to account for the user clicking on the ad.
