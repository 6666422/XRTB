Open Source

This is an implementation of a private exchange, for use with RTB bidding engines to sell ad space on web pages. The Private Exchange is written in Java 1.8 and is completely open source. In this demonstration implementation, this private exchange will operate with the RTB4FREE bidding engine that is also hosted on this site.

RTB Private Exchange

An open source private exchange for selling ad-space on web pages. This private exchange implements openRTB mobile bidding platform. On one side of the private exchange are the web pages with ad-space for sale.

On the other side of the exchange are RTB bidding engines. Anyone loading a web page with the Private Exchange 'for-sale' code in it contacts the Private Exchange with the demographics of the user that is loading that page.

The Private Exchange sends the bid requests to the RTB bidding engines. The RTB Bidding engines respond with bids, Private Exchange holds the auction, notifies the winner, and gets the impression.

The Private Exchange sends the impression to the web page, and the user sees the ad that won the bid.

RTB Bidding Engine

An openRTB mobile Real Time Bidding engine (2.2) written in Java 1.8. This bidder supports multi bid farms, and includes full constraint checking on bid requests. The bidder also supports non-standard constraint checking and is easily extendable to handle non standard targeting.
